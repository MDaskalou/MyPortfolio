Booking System for a Hospital

This is the biggest project I'm working with so far. The purpose of this system is to create an efficient booking system for departments within the Sahlgrenska hospital in Gothenburg. The main goal is to reduce the need for nurses to call the department every time they need to make a booking, and instead allow them to use this system. It was a large-scale project where I learned a lot — including authentication, MediatR, controllers, services, APIs, and much more.
